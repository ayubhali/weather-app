## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add your OpenWeather API key:
   ```
   VITE_APP_ID=your_api_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- React
- Vite
- OpenWeather API
- CSS3 with Flexbox
